Courier

Courier is stateful, typed programming language interpreted in OCaml. The language includes a variety of non-standard features, such as registers, a 'default' function, user-configurable type-error handlers, and a wide variety of other aspects of state which can be set and changed in the language. The project was a course project for the course CS 4110: Intro to Programming Languages and Logics.
